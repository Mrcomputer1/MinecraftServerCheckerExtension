    How To Use

    Is [] () Online?

    Is [SERVER IP] (SERVER PORT) Online?

    If it returns 1 it is online if it returns 0 it is offline

    Motd of [] ()

    Motd of [SERVER IP] (SERVER PORT)

    It should return the message or it will return 0

    Online players of [] ()

    Online players of [SERVER IP] (SERVER PORT)

    It should return the online player count of 0

    Max players of [] ()

    Max players of [SERVER IP] (SERVER PORT)

    Returns max players or 0

    Server Software of [] ()

    Server Software of [SERVER IP] (SERVER PORT)

    Returns server software or 0

    Is [] () [ v]

    Is [SERVER IP] (SERVER PORT) [VERSION v]

    Returns 1 if it is that version or 0 if it is not that version it will return -1 if it is offline

    Get Version [] ()

    Get Version [SERVER IP] (SERVER PORT)

    Returns 1.8.3, 1.8-pre3, 1.8-pre2, 1.8-pre1, 1.7.10-1.7.6, 1.7.5-1.7.1pre or 0
